# EcoEnergy Tracker 🌍⚡

EcoEnergy Tracker — это веб-приложение на Python, созданное для хакатона на тему глобализации и устойчивого развития.  
Проект помогает пользователям отслеживать и анализировать потребление электроэнергии в доме или офисе, а также показывает способы уменьшения расхода энергии.

## 🎯 Цель проекта

Главная цель проекта — повысить осознанность людей в вопросах энергопотребления и помочь сократить лишние траты электричества.

Проект связан с глобальными проблемами:
- изменение климата;
- загрязнение окружающей среды;
- перерасход ресурсов;
- энергосбережение.

---

## 🚀 Возможности

- 🔌 Подсчёт расхода электроэнергии
- 💡 Анализ потребления света и бытовых приборов
- 📉 Рекомендации по снижению энергозатрат
